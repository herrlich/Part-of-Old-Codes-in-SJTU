        ? ??
        ? ?   ? 中共中央政治局委员、国务委员刘延东同志视察上海交大申瑞民教授领衔的移动学习成果展示
         
           2011年12月24日，全国继续教育工作会议开幕。中共中央政治局委员、国务委员刘延东在教育部部长袁贵仁、副部长鲁昕的陪同下，来到会议中心展区的上海交大继续教育学院移动学习成果展台，视察交大申瑞民教授领衔的移动学习成果展示。鲁昕向刘延东介绍了上海交大申瑞民教授领衔的移动学习研究和推广应用工作，上海交大E-learning   Lab工作人员向刘延东展示了和宁夏大学、西藏大学的移动学习直播课程现场播放与实时交互。上海交大移动学习高水平研发工作以及在宁夏和西藏的高校、职校和农牧边防地区的推广应用得到了刘延东和教育部领导的肯定。

                [发布时间]: 2012年10月25日

           　　2012年10月22日晚19时08分，上海交通大学原党委书记邓旭初同志因病不幸在上海华东医院去世，享年92岁。（讣告全文）
    消息传来，交大师生深感震惊和悲痛。校党委书记马德秀、校长张杰第一时间致电邓旭初家人表示关切和悼念。在邓旭初同志住院期间，学校领导及有关部门多次前往医院看望。
　　10月22日、24日，上海交通大学党委书记马德秀、校长张杰怀着沉痛的心情前往吊唁，并代表学校和全校师生向邓旭初家属表示关切和慰问。校党委常委李建强、党办、校办、党委组织部、党委宣传部、人力资源处、老干部处等部门负责人一同前往。
　　邓旭初同志简朴的家里摆满了花篮，校领导向邓旭初同志的遗像鞠躬致礼，并与邓旭初的弟弟及女儿、女婿等家属亲切握手，表达关心和问候。校领导表示，获悉邓旭初不幸逝世的消息后，学校在第一时间就组织有关部门协同家属商榷后事工作并协助办理相关事宜。邓旭初同志在改革开放初期就担任上海交大党委书记，期间他率先试行高校管理改革，破冰中国高校的国际交流之旅，开国内高校引进外资的先河，一系列改革措施推动了中国高等教育改革。学校成立邓旭初同志治丧办公室，党委书记马德秀任主任、校长张杰任副主任，会同家属及有关部门办理好邓旭初同志后事。
　　邓旭初的家属表示，邓旭初对交通大学怀有深厚的感情，交通大学也一直对邓旭初及家属给予无微不至的关心，他们对学校一直以来的关心和校领导的看望慰问表示感谢。

            上海交通大学对口支援宁夏大学第十三次联席会议举行[图]
      
      
            
                [发布时间]: 2012年10月26日
        
        
        
            
                 
                            [推荐新闻]     [我要纠错]
                 
                            字号：[大 中 小]
                     
                 
            
        
        
            [责任编辑]: 卢思语
        



          
        
           　　10月22日，上海交通大学党委书记马德秀会见宁夏大学党委书记齐岳一行，对双方进一步的合作进行了广泛深入的探讨。10月23日，上海交通大学对口支援宁夏大学第十三次联席会议举行。校长张杰、副校长黄震、党委常委李建强、BioX-中心主任贺林，宁夏大学党委书记齐岳、副校长谢应忠、王春秀、党委常委王玉炯出席。会上，黄震、谢应忠分别代表上海交通大学和宁夏大学作年度工作总结报告。双方签署了《上海交通大学对口支援宁夏大学重点工作实施备忘录》。

　　张杰在讲话中表示，两校友谊延续了十多年，近几年，随着合作的深入，双方联系的纽带不断加强。在人才培养、招生、学科建设、科学研究、干部交流等各个方面取得了丰硕成果。宁夏大学近年来的高速发展为双方进一步的合作创造了新的契机。双方在人才培养、国际交流、“校、地、企”合作、高端培训等方面形成了独有的特色。在今后的合作中，一定要进一步加强合作的特色，并形成优势。要在服务国家需求上谋求更多的携手共赢，同时加快世界一流大学建设的交大也切实需要宁夏大学在各方面的的支持和帮助。

　　齐岳在讲话中表示，宁夏大学在交大对口支援的十年间获得了长足的进步。这种支持是实际的、有效的、得力的。交大输送到宁夏大学的干部和教师思想有活力，工作有热情，对宁夏大学的发展充满了感情，为宁夏大学的干部、教师转变观念、解放思想起到了关键性的作用。今后，双方的合作要进一步探讨合作的新途径、新举措、新办法。要围绕西部开发中宁夏的区域优势和国家协同创新举措的实施开展工作。他还建议双方在人才培养的合作模式上，在学术著作的出版上以及在两校师生交流访问空间的拓展上进一步深化合作。


　　期间，上海交通大学副校长郑成良会见了宁夏大学副校长谢应忠、王春秀一行，并主持了两校各相应部门之间的对口合作交流活动。

            [青春足迹]李天霓：用我的整个青春去经历[图]
      
      
            
                [发布时间]: 2012年10月26日
        
        
        
            
                 
                            [推荐新闻]     [我要纠错]
                 
                            字号：[大 中 小]
                     
                 
            
        
        
            [责任编辑]: 张舒尧 思源
        



          
        
           　　自画像：李天霓 媒体与设计学院 大三
　　我既有大众化的爱好，比如摄影、旅游、阅读；又有小众化的特长，比如扬琴。我有斗志，立志成为现代女性，当厚德载物、自强不息；又有情怀与理想，追求永葆朴素、自由、看得见、听得见的自己。

　　艺术节形象代言人：艺术节的风吹上我的脸
　　2012年3月，上海交通大学国际大学生文化艺术节进入筹备期。主办方联系了我，希望我能够加入艺术节团队，担任代言人。大家在一起聊高雅艺术进校园、谈让民族音乐走进校园，回想起来仍印象深刻。短短两周时间，喷绘、网络平台攻势渐起，而最好的效应，是陆续有人问世间扬琴为何物，或恍然大悟艺术团民乐队的存在。怀想那年，我5岁，已是幼儿园小班。每日准点被按在琴凳上，一下一下别扭地敲着与我毫不相干的音符。彼时并不流行艺术的叫法，只道是修养。当求学已成往事，艺术尚是前车的尾灯，夜未深时不企盼以及常疑远去的存在。大学以后，2011年代表上海交通大学艺术团参与魅力中国专场音乐会演出，2012年代表艺术团赴德国、比利时各城市孔子学院参与龙年春节慰问演出。有活蹦乱跳的孩子趴在矮矮的台阶上看我弹琴，那股喜欢劲用余光就轻易获取，音乐从我手下的变成别人耳朵里的。分享艺术情结成为一种习惯，我这辈子也许最美好的习惯。
　　创意嘉年华工作人员：不经意间爱上“文化”这件小事
　　我大学就读的专业，也许有人闻所未闻。文化产业与管理是什么，我亦费不少精力与时间去告诉自己，然后解释给别人听。必修课程清单上罗列诸多文化课程，如文化行政、文化消费学概论、文化商务写作、文化产业学等。专业学术一度在我看来是把艺术欣赏渡成温柔一刀，浅浅慢慢地消磨对文化的品茗。幸而，接触不少校园文化建设的一线工作人员，依恃代言人的身份，探手即是嘉年华、开幕式。像搭积木，一块块地构筑文化石堡，贪恋不全是恢弘效应，因为大多基石同时垒在我心里。活动当天，隐匿于慕名而来的人群，没有穿工作服。抱着娃娃的妈妈凑过来向我讨个气球，旋即讶异我是体验券里走出来的人，忙不迭教着喊我姐姐，还表示未来也让孩子学扬琴、做代言人。我不知道10年、20年后还是否有另一个艺术节，唯搬出单反记录这段史。我忽然意识到，若非总怀好奇心，若非抱创意之大枕，文化决议或只是纸上谈兵。在一个你也许不知道的专业“落户”，经历校园新鲜出炉的热乎事，一边耗大把时间观察、阅读，沉的夜缀着星辰是美丽，灯光簇亮的一瞬回归沉淀，我方才紧贴所谓生活、所谓文化。
　　青年论坛组织者：用我一整个青春去经历
　　微博互动时常遇人说我是一个经常可以看见的人，那大概缘于一句话：在大学的时候，你总要爱一件事，爱一个组织，那才是完整的。一般情况下，我享受陀螺式鞭挞着的奋斗，体验截然不同的角色或任务，甚至挫败时嚼着正面力量的话拍拍身上的土继续让身体在路上。上海交通大学国际大学生文化艺术节高校青年文化论坛，组织、策划、实施、奔走，安排妥帖行程已非易事，遑论邀请C9高校代表全程参与。忆起，全然忘却熬的夜有多深或者筹备期间调配的细枝末节，只记得患了不大不小的“强迫症”，会时感包里的手机在震动。若接的电话不是顺顺利利报喜讯，还不得以一己之情绪影响工作人员。那时就开玩笑，说活动一场让我“老”了好几岁。
　　值得吗？我不愿意回首来时的路，见一个只敢想不敢努力、只愿停留不愿开拓的自己，那只能选择相信在合适的年龄做合适的事，青春终是无悔。

            《远东国际军事法庭庭审记录》编纂出版工程启动[图]
      
      
            
                [发布时间]: 2012年10月26日
        
        
        
            
                 
                            [推荐新闻]     [我要纠错]
                 
                            字号：[大 中 小]
                     
                 
            
        
        
            [责任编辑]: 卢思语
        



          
        
           　　10月24日，由上海交通大学和中国国家图书馆联手实施的100卷本《远东国际军事法庭庭审记录》编纂出版工程，在北京正式宣布启动。这是国内首次公开出版研究远东国际军事法庭审判的第一手文献资料。教育部副部长杜占元、文化部副部长杨志今、新闻出版总署副署长邬书林、中国社会科学院院长助理郝时远、上海交通大学校长张杰、中国人民大学党委书记程天权、国家图书馆馆长周和平出席启动仪式。张杰校长和周和平馆长共同为该项目启动揭幕。

　　杜占元在讲话时表示，教育部会持续支持该项目研究和平台建设。据不完全统计，西方学术界出版了五十余种相关学术著作，日本出版的学术著作和通俗读物不下两百种。相较而言，中国学者对东京审判的研究还比较薄弱，专业研究人员数量不足，基本史料严重缺乏。 
　　杨志今希望，参与这次编纂出版工作的有关单位想方设法，尽快实现海外相关文献的回归，并加快文献整理和影印出版进度，促进更多成果问世，以留下完整的历史记忆，同时加强对“东京审判”文献的研究，深入总结历史的经验和教训。此外，还应加大宣传力度，使相关研究成果为学界和社会公众所了解和使用，充分发挥这项工作的社会效益，使之成为爱国主义教育的生动教材。
　　邬书林指出，东京审判是二战结束后对日本28个甲级战犯的国际审判，代表了国际正义力量对法西斯邪恶势力的审判，彰显了理性、公平和正义。中国是受日本军国主义侵略伤害最深的国家之一，无论是学界还是出版界，都有责任对这段历史进行全面深入研究，让更多人了解东京审判。
　　周和平在作“《远东国际军事法庭庭审记录》编辑出版工作报告”时说，《远东国际军事法庭庭审记录》将展现东京审判的全过程，这是通向远东国际军事法庭审判、消除歧见与争论的有效途径，也是驳斥“东京审判史观”、“远东国际军事法庭非法论”等日本右翼保守势力歪曲和否认侵华史实言论的强有力的武器与佐证，将有助于在整理、保护珍贵的历史文献、推动对东京审判的关注与研究的基础上，为解决现实争端提供历史依据与法理依据。2011年，国家图书馆与上海交大合作建立东京审判研究中心，研究人员逐页检查了近五万页馆藏庭审记录，今年4月，又派员专赴美国深入挖掘史料，搜寻补得7000余页，基本补齐庭审记录原始文献，为《远东国际军事法庭庭审记录》的编纂出版奠定了基础。
　　张杰指出，《远东国际军事法庭庭审记录》的出版标志着中国在东京审判研究的领域迈出了极为重要的一步，是上海交通大学东京审判研究中心与国家图书馆自今年三月正式开展合作以来的第一项成果。远东国际军事法庭审判，影响了整个东亚的历史进程，是战后日本和远东国际关系新格局的起点，某种程度上说，东亚国际关系中的一些重大战争遗留问题都可以追溯至远东国际军事法庭审判。一直以来，东京审判的正面意义正在不断地被有意或无意地淡化和漠视，时至今日，中日两国的关系仍然时时被历史问题所困扰。东京审判从法律上对日本军国主义在中国犯下的罪行进行了清算，然而仍有不少遗留问题没有得到彻底解决。面对这样一场对中日两国如此重要的国际审判，中国绝不能在相关的研究领域中缺席和失语。张杰强调，《远东国际军事法庭审判庭审记录》仅仅是双方庞大出版计划的开始，相信随着更多档案文献和学人著作的陆续问世，中国的东京审判研究一定可以取得长足的进步，为理顺中日两国关系提供重要的理论支撑和智力支持。
　　启动仪式结束后，《远东国际军事法庭庭审记录》编纂委员会召开专家座谈会，征求各方专家意见，北京大学、南京大学、中国社会科学院、首都师大、南京师大、江苏社会科学院等专家出席。为配合编纂出版启动仪式，在国家图书馆还同时举办了“历史的审判――馆藏远东国际军事法庭审判历史文献展”，展出了馆藏远东国际军事法庭庭审记录原件缩微品、民国时期记录日本侵华罪行的珍贵期刊报纸原件、世界各国对二战日本战争罪行和远东国际军事法庭的研究文献等400余件，曾参加东京审判的国际大法官倪征的后人送展所藏部分庭审记录及判决书原件。这些文献展示了日本军国主义意图奴役中国人民，掠夺中国资源的昭然野心和滔天罪行，直接驳斥了现今日本右翼势力歪曲和否认侵华史实的言论。
　　据悉，《远东国际军事法庭庭审记录》将由国家图书馆出版社和上海交通大学出版社联合编纂出版，100卷将于明年11月东京审判结束65周年之际出齐，该项目版权业已被剑桥大学出版社引进，现已列入“剑桥中国文库”系列。
　　我校东京审判研究中心主任陈兆奇教授会后还接受了中央电视台等主流媒体的采访，校办、文科处、出版社等相关部门负责人参加了启动仪式及座谈会。

            [学者笔谈]王一飞：半个世纪的大学情缘[图]
      
      
            
                [发布时间]: 2012年10月27日
        
        
        
            
                 
                            [推荐新闻]     [我要纠错]
                 
                            字号：[大 中 小]
                     
                 
            
        
        
            [责任编辑]: 卢思语
        



          
        
               [编者按] 继2011年上半年推出“身边的感动”系列报道受到广泛好评后，从2011年10月起，我们推出了新栏目“学者笔谈”。本栏目将陆续推出一批我校有影响的学者，重点展示他们在人才培养、科学研究、服务社会和文化传承与创新等方面的观点和见解、思路和做法及理论和实践，旨在弘扬科学精神，激荡人文情怀，回归学术本位，浓郁学术气象，全面提升交大学术的影响力和传播力。 
　　■ 每当我登上教室讲台，就会精神抖擞，思维活跃，充满幸福感与成就感，我可以自豪地说：“三尺讲台是我的终生岗位”。
　　■ 一流大学的建设不是一蹴而就的，一流大学的建设不是一个名词，而是一个动词，并且是一个进行式动词。
　　■ 转化医学是当前医学研究与健康服务的主要战略之一，医学教育必须顺应潮流，未雨绸缪，医学教育改革迫在眉睫，势在必行，这是全球医学教育界面临的共同挑战与机遇。

　　我出生在一个教育世家，家庭的核心成员也都与教师结下了不解之缘。自从1957年我考入上海第二医学院并于1962年留校至今，已经整整五十五年了。这期间，我担任过十年大学校长以及六年联合国官员，但最让我终生依恋和难以舍弃的岗位是“传道，授业，解惑”的教师。每当我登上教室讲台，就会精神抖擞，思维活跃，充满幸福感与成就感，我可以自豪地说：“三尺讲台是我的终生岗位”。
　　母校上海第二医学院（现称上海交通大学医学院）成立于1952年，今年欣逢母校成立六十周年。六十年是一个甲子，五十年是半个世纪，但在人类历史的长河中，这只是短暂的一瞬。我在母校已经学习和工作了五十五个寒暑春秋，我熟悉这里的一草一木，我铭记这里的一情一景。在庆祝母校六十华诞之际，我心潮澎湃，感慨万千。 
　　大学最宝贵的财富是学生
　　什么是一所大学最宝贵的财富？根据一般的评价标准，一所一流的大学应该包括以下主要标志：宽敞绮丽的校园，蔚为壮观的建筑，汗牛充栋的藏书，设备精良的仪器，蜚声中外的大师，声名卓著的学科，举世瞩目的成果，高瞻远瞩的视野，新颖独特的理念，精辟深邃的精神。但我认为贯穿以上几项标准并作为共同基石的是学生。换言之，一所大学最宝贵、最重要的财富是学生。原因在于：第一，大学的首要任务是传授知识和培育人才。学生是一所大学最重要的“产品”，是一所大学的“品牌与名片”，是反映一所大学水平最直观的标杆。第二，学生是一个广大的群体，又是一个流动的群体。他们来自天南海北，在大学校园苦读三年五载，然后走向五湖四海，把一所大学的思想播撒到四面八方，向社会公众展示一所大学的形象。第三，一所大学的文化、精神、传统、胸怀、视野与理念是经过岁月沉淀而成的，这些独特的文化与精神需要有人去继承，去发扬。学生既是大学文化与精神的传承者，又是发扬光大的生力军。
　　我清晰地记得，入学后不久，当时主管医学教育的章央芬副院长在一次全校大会上明确提出“二医的中心工作是一切为了学生，要千方百计地提高教学质量。”这不是一句空泛的口号，而是切切实实的真抓实干。当时曾采取了几项十分有力的措施：
　　一是根据培养目标，调整教学内容，减少重复环节，鼓励临床医师参加基础医学教学。当时王振义教授就从瑞金医院内科来到基础部筹建并领导病理生理教研室。
　　二是各科的主要教授必须为本科生授课。我回顾了一下我的求学经历，八门基础医学主干课程有六门均由教研室主任教授亲自上课，其中生理学从大班讲课到实验指导均由张鸿德主任教授一人承担。
　　三是加强第二课堂教学，因材施教，调动学生的主观能动性，着力于学生的能力培养和素质提高。我是这个措施的直接受益者。从一年级开始我就积极参加学生课外科研小组活动，先后进行了农村儿童钩虫病流行病学调查、胃癌175例临床病理分析及肝性昏迷发病机制等研究，我还在全校学生科研报告会上宣读论文，受到黄铭新教授的赞扬。余贺教授专门开设讲座，为学生讲授如何阅读学术论文，他鼓励我们多看文献，学会将知识综合归纳，学会驾驭知识。在他的感召下，我利用了几个暑假，以人体的八大器官系统为框架，把已经学过的基础与临床医学课程，进行知识的重组与整合。当时还没有电脑，我就在不断地抄写、描画、拼接和整理的过程中，经过系统回顾与思考，逐步学会了把浩瀚的知识和信息分门别类地储存到自己头脑中的不同区域，需要时还能随时加以提取、重组、演绎与拓展。
　　四是为了保证教学质量，必须贯彻三基（基础知识、理论与技能）与三严（严格、严肃与严厉）的精神，注重学生的德智体全面发展。章央芬院长曾语重心长地对莘莘学子说：“在大学里既要学知识学技能，还要学会做人，一定要又红又专。”我不禁想起英国前首相丘吉尔曾经说过的一句名言：“大学不仅要传授知识，大学更应当传授智慧。大学不应当满足于培训技术，大学应当致力于学生的人格锻造。”我认识到大学的使命应当是让智慧与人格深深地烙刻在每个学生的“人生光盘”上，让学生都能终生受益。
　　一流大学的两个支柱：一流学科和一流人才
　　母校成立于1952年，是由原来的圣约翰大学医学院（美国宾州大学医学院上海分校）、震旦大学医学院及同德医学院合并而成。这期间融合了来自美国、法国及德国与日本等不同流派体系的医学理念与风格，因此先天就具备五湖四海、相互包容、取长补短与海纳百川的优势，形成了不拘一格、开拓进取、敢为人先、面向世界的传统。正是由于这个学校吸纳了一大批上海的名医加盟，因此临床技艺精湛，创造了一个又一个的“中国第一”与“世界奇迹”：国内第一例心脏二尖瓣分离术、国内第一次用阿托品治疗锑剂中毒引起的阿斯综合征、国际首例抢救大面积烧伤患者、国际第一例断肢再植、直至转化医学的典范――诱导分化治疗急性早幼粒白血病，为此王振义院士获得了2010年国家最高科学技术奖。
　　在回顾这些光辉业绩的同时，我们应当认真分析并总结出一些规律，以期指导母校今后的建设与发展。纵观世界一流大学的成长轨迹，结合母校发展的经验与教训，我认为一流大学不是自封的，也不能按“计划打造的”，所谓一流大学的标准主要是看其是否有一批基础扎实、立足前沿的学科；是否拥有一批大师级的学者；是否能在全球吸引高质量的学生；是否致力于将教学与科研有机结合，对科技发展做出重大贡献；是否对社会与人类文明产生重大影响；是否建立起一套科学与民主的决策和管理机制；是否建立起与政府、企业和公众之间良性互动的渠道；是否在国内外学术界占有举足轻重的地位；是否形成大学自身独特的文化传统与精神；是否具有公正、诚信、睿智、博爱的公众形象。上述十条才是世界一流大学的内核，其中一流学科与一流人才更是两大支柱。
　　1958年母校成功抢救严重烧伤患者邱财康（烧伤总面积89%，其中Ⅲ度烧伤23%）就是依靠一流学科与一流人才有机结合创造的世界奇迹。当时国际上的定论是“烧伤面积超过80%无法治疗”。但在党委与院领导的统一指挥与精心安排下，组织了外科、内科、感染与免疫、整形及护理等各科专家，各尽所能，通力合作，历经三个月的奋战，终于安然渡过了休克、感染及焦痂处理等重重难关，利用噬菌体控制了绿脓杆菌感染，采用混合植皮技术使创面得以完全愈合。以史济湘及杨之骏教授领衔的多学科合作团队，总结出的严重烧伤治疗规范被誉为“中国公式”。
　　由于众所周知的原因，母校在十年文革浩劫后出现了学科凋零与人才断层的严峻局面，学校领导与全体员工充分认识到学科建设与梯队建设的重要性，断然采取了重点学科建设、学科群建设规划、创建新兴学科、破格晋升、重点学科带头人和学术骨干培养计划、配套成组出国培训、开辟国际交流与合作渠道等措施，尤其是通过“211工程”建设规划，终于迎来了人才辈出与学科建设欣欣向荣的新局面。
　　所谓“配套成组出国培训”的模式，是将不同的专业人员依据学科建设的需要组织成一个团队，在课题负责人的领导与统筹下，共同为攻克一个研究课题、创建一个新的学科或建立一支高水平的学术梯队，有针对性地出国培训。由于目标明确、分工合理、配套成“龙”，因此成效卓著。例如，丁文祥教授领衔的小儿心胸外科，在美国民间健康基金会的支持下，有计划地“派出去，请进来”。经过几年的努力，很快掌握了能体现一个国家小儿心脏外科技术水平的各类紫绀型先天性心脏病的手术和危重症监护技术，而且研制出一整套适合中国国情的专用医疗器械和设备，从而使这个学科跻身世界先进行列。并在此基础上建成了堪称亚洲一流的上海儿童医学中心。
　　一流大学的建设不是一蹴而就的，一流大学的建设不是一个名词，而是一个动词，并且是一个进行式动词。我们离开世界一流大学的标准还有很大的差距，还需几代人的不懈追求和努力奋斗。
　　打破六堵围墙，让大学活力充分绽放
　　教育落后于社会发展是全世界普遍存在的现象。因为教育是在为未来培养人才。就以医学教育为例，今天的医学生是明天的医生。可是没有人能确切知道未来社会究竟是什么样，需要什么样的人才，需要什么样的医生，这是全世界医学教育面临的共同挑战，当然也是一种机遇。因为哪所大学能比较准确地把握未来医学的发展趋势和对人才的需求，这所大学就能占领未来高等医学教育的制高点。
　　有鉴于此，我与我的同事们，从2002年开始，在交大医学院进行了小规模的医学教育改革试点，其要点如下：
　　一是，目标是根据我国医学卫生事业发展的需求，培养一批基础宽、能力强、素质高、复合型的高层次临床医学专门人才。
　　二是，依托上海交通大学的优势，拓宽学生的自然科学与人文科学基础，加强通识教育及强化现代化信息技术及外语能力培训。
　　三是，让学生早期接触临床，在为病人提供基本医疗服务的过程中树立救死扶伤的责任感，学习如何建立良好的医患关系。
　　四是，改变以往基础医学与临床医学教学绝然分开的模式，使基础医学与临床医学教育相互交织，几次反复，不断深化，螺旋式上升。
　  五是，打破传统的以学科为中心的教学框架，在基础医学教学中开设“医学导论”，并采取人体器官系统的整合课程，在临床教学中选取若干重要疾病进行多学科的综合培训。
　　六是，采用以教师为主导，学生为主体的主动参与的学习模式，包括PBL（以实际问题为基础的学习）、CBL（以临床病例为基础的学习）、RBL（以研究探索为基础的学习）等，着重培养学生的自学能力、动手能力、解决问题能力、交流沟通能力及分析归纳和批判性思维能力。
　　总之，大学必须认真思考如何更好地为社会可持续发展与人类文明进步主动做出贡献，大学应当成为社会中最具有蓬勃生机的一员，大学应当打破以下六堵围墙，才能使大学活力绽放：
　　● 打破学科之间的围墙，打破学院之间的围墙，多学科的交叉融合和互补合作具有无限的生命力；
　　● 打破正式课程与课外活动之间的围墙，学生应当到第二课堂和到工作现场去体验，去实习，以培养他们的实践能力、团队精神及应变能力；
　　● 打破大学与企业间的围墙，所谓产学研合作机制，就是将基础（Basic）、 中试（Transitional）与转化（Translational）3种研究与开发加以协调与整合；
　　● 打破大学与政府决策部门之间的围墙，大学应当成为政府决策的思想库与智囊团；
　　● 打破大学与公众之间的围墙，包括打破大学与中小学之间的围墙。因为一个社会中公众的知识素质会直接影响高等教育质量。但必须强调的是，大学不能以教育者自居，教育从来都是双向的，大学要学会倾听和体恤， 要学会汲取和反哺；
　　● 打破中国大学与国际学术界之间的围墙，所谓与国际接轨，不是照搬发达国家的模式，核心是交流、交融、共振与共鸣。
　　作为母校六十华诞庆典的一部分，我主持了由世界医学院校联盟与交大医学院共同召开的国际医学教育研讨会，主题是：转化医学时代的医学教育，与会的有几十位国内外知名医学院校的专家、医学研究机构与企业界的代表以及政府官员。这次会议的一个重要共识是：转化医学是当前医学研究与健康服务的主要战略之一，医学教育必须顺应潮流，未雨绸缪，医学教育改革迫在眉睫，势在必行，这是全球医学教育界面临的共同挑战与机遇。美国著名的教育家Howard Gardner指出，我们的未来需要五种人：受过专业培训并训练有素的人；善于归纳与整合各种信息的人；有独立见解并能不断开拓创新的人；尊重别人并有团队合作精神的人；有崇高道德和深厚文化底蕴的人。
　　回顾半个世纪的大学生涯，我曾拥有过多种身份――学生、教师、系主任、校长、联合国官员，但我最重视的身份是“二医人”和“交大人”。这所大学横跨三个世纪，具有深厚的文化历史沉淀，并赋以开拓创新敢为人先的历史重任。我们都是这条世纪长河中的一个水滴，我们都立志为母校的发展壮大不断添砖加瓦。
　　我今天仍然站在我熟悉的母校讲台上。在我看来，三尺讲台虽小，但它传承过去，又连接未来。站在三尺讲台上，传授知识，培养人才，站在三尺讲台上，启迪思维，激励创新。每一个教师都是一个摆渡者，一个引路人，每一个教师都应当在攀登科学高峰的同时舍得把自己的肩膀作为科学人梯上的一个台阶，让莘莘学子踏着这个台阶向着更高的山峰攀登。只有这样，我们的国家与民族才会迎来人才辈出的崭新局面；只有这样，人类社会才有了可持续发展的强大动力与无穷源泉。
    学者小传

            诺贝尔物理学奖获得者Theodor W. H?nsch教授做客大师讲坛[图]
      
      
            
                [发布时间]: 2012年10月26日
        
        
        
            
                 
                            [推荐新闻]     [我要纠错]
                 
                            字号：[大 中 小]
                     
                 
            
        
        
            [责任编辑]: 卢思语
        



          
        
           　　10月25日下午，诺贝尔物理学奖获得者、德国慕尼黑路德维希-马克西米利安大学实验物理学教授、德国加尔兴马克斯-普朗克学会量子光学研究所理事Theodor W. H?nsch教授做客第十二期上海交通大学大师讲坛，以“Passion for Precision”为主题，阐述了实验物理学研究的重要性，以及理论与实验之间看似微小的差别可能带来的概念上的重大突破。本次讲座由上海交通大学和霍尼韦尔公司联合主办。校党委常务副书记苏明教授和霍尼韦尔公司中国区总裁盛伟立出席活动并致辞。全校600多位师生在陈瑞球楼100号报告厅聆听了这场精彩的讲座。

　　苏明对Theodor W. H?nsch教授做客大师讲坛表示热烈欢迎，对霍尼韦尔公司在人才培养、科研合作等方面对交大的支持表示感谢。苏明介绍了H?nsch教授在光学领域的突出贡献。H?nsch教授因其对基于激光的精密光谱学发展所做出的杰出贡献，与Roy J. Glauber和John L. Hall一起荣获2005年诺贝尔物理学奖。苏明表示，将世界最前沿的科学带进校园，让著名学者与青年学子近距离接触，意义深远。H?nsch教授的演讲将给交大师生带来极大启发，激发学生在科学领域追求卓越的兴趣。

　　盛伟立表示，非常高兴携手诺贝尔获奖者再次走进交大校园，为未来的科学家和工程师们提供与当代大师面对面交流的机会。他希望通过“霍尼韦尔卓越科学与工程计划”这一形式独特的全球项目，为上海交大等著名高等学府在开拓学生视野、培养科学兴趣等方面贡献一份力量。盛伟立强调，上海交通大学是霍尼韦尔在中国最重要的合作高校之一，双方的强强合作必将推动科技创新和中国优秀人才的培养。

　　Theodor W. H?nsch教授在报告中介绍了物理学的基本规律。H?nsch教授谈到，爱因斯坦的相对论就是基于精密测量。无独有偶，巴尔默氢原子光谱较为简单，却为解密量子物理学的奇特法则提供了一把钥匙。通过对其进行精密光谱学研究，可以得出重要物理常量的精确数值，并较为精确地测试基本物理学规律。
　　H?nsch教授重点介绍了激光频率梳。激光频率梳本质是非常强的激光，在测量频率时是一个非常有用的工具。十年前正是激光频率梳的发明为人类提供了以极高的精度计算光波脉动的方法，使得氢内的共振频率测量精确到小数点后15位成为可能。H?nsch教授进一步列举了激光频率梳的部分应用以及最新的物理学趋势。

　　H?nsch教授分享了自己的科研经历，勉励交大学子不仅要学习书本知识，更要学会主动探索未知领域。最后，Theodor W. H?nsch教授受赠礼物并为讲坛留下了“大师手印”。 
　　Theodor W. H?nsch教授凭借其在激光光谱学领域的开创性贡献而闻名世界。他的早期工作包括发明第一台可调谐窄带脉冲染料激光器，开创广为使用的无多普勒激光光谱技术，并提出激光冷却原子气体的建议。近年来，H?nsch教授率先利用革命性的频率梳技术来测量超短脉冲光的频率。在中性冷原子的量子物理学领域，H?nsch和他的同事们用光实现了首个二维和三维原子晶格束缚。

    中国医药连锁企业国际电子商务EMBA总裁班隆重开班[图]
   
          
			   2012年9月1日09:00，由上海交通大学继续教育学院与北京搜药传媒联合开办的《中国医药连锁企业国际电子商务EMBA总裁班》在上海交通大学徐汇校区教三楼隆重开班，上海交通大学继续教育学院副院长刘天鸣和搜药传媒董事长郭亚洲出席了开学典礼。
本次开学典礼由上海交通大学继续教育学院电子商务项目主任曾显竣主持，对上海交通大学国际电子商务EMBA总裁班的总体课程做了表述，让每位来到班级的同学对上海交通大学的课程体系都做了一个深入的了解,并将全力打造电子商务高端品牌课程，服务好每一位学员，让学员在交大学有所获、事业腾飞；接下来由郭亚洲董事长、学员代表德生堂药业董事长龙岩、刘天鸣教授致辞；刘天鸣教授回顾了上海交通大学继续教育学院的发展历程并阐述了学校的校训校风，且提出了对本次培训的期望；学员代表德生堂药业董事长龙岩发表非常感谢上海交通大学与搜药传媒联合举办的这次电子商务EMBA总裁班，我们是很传统的医药行业，电子商务作为新兴的商业渠道平台，已经越来越发展成为一个相对重要的市场，未来的商界也势必是熟练电子商务应用的天下。搜药传媒郭亚洲董事长本着对电子商务发展趋势的看好，并正是在这样的背景下，与百年名校上海交通大学联合举办本次培训班，学校整合卓越师资、设计全新课程体系，精心为此医药行业打造《中国医药连锁企业国际电子商务EMBA总裁班》，真诚为各企业的电子商务转型指点迷津！
最后，参加本次开学典礼的领导、学员们在上海交通大学共同合影留念，开学典礼宣告圆满结束，上海交通大学《中国医药连锁企业国际电子商务EMBA总裁班》正式开班授课。
 

 
上海交通大学继续教育学院副院长刘天鸣教授致辞中

培训班学员感言:突出理论 增加实务


2009年9月29日 15:35




　　尽管事先组长关照说希望能多提些意见，但是我还是想先说，自己在这次培训中受益匪浅。主要有以下三点： 
　　首先是能够一次性听到这么多优秀的老师和学者的讲课是这次培训令我感到最兴奋的事，出乎我的预想之外。我今年刚从上海外国语大学新闻传播学院毕业，大学期间就陆续接触到很多复旦新闻学院老师的著作。一直希望有机会能来听，如果没有参加这次的培训，我想我也不可能有机会坐在复旦大学的校园内面对面地聆听这么多老师的授课了。一些复旦的同学告诉我，他们作为本科生很少有机会听到像李良荣等老师的讲课。所以能有这样的一次培训机会真的感到很幸运。 

　　其次，这次培训的课程也十分多元化。既有偏理论性的课程，也有指导实际编辑运用的课程，既有宏观的国情市情舆情的介绍，也有微观的诸如广告、网络编辑等内容。让我们从多领域，多角度，多层次地对网络新闻与信息业务有更全面和深刻的了解。 

　　从我的个人的角度而言，这次培训所传达的一些知识与我的工作职能非常贴近。比如东方网采编中心高国营总监讲解的网络编辑业务专题，对我们从事新闻编辑也好，传播信息、论坛发帖也好，有着非常直接的指导作用。 

　　由于不少学员除了新闻编辑外，还负责网络的综合运营，论坛的管理等工作，我想在今后的培训课程中，可增加网络综合管理的培训，丰富培训的实务，让学员在实务方面多进行沟通交流，并有综合性的提高。 

　　参加培训的第三点受益就是，能够借此机会与来自不同网站的同僚进行沟通，增进相互了解，相互学习。在沟通过程中，我也又在网上找到了很多好的网站。 
　　另外，这次班级还分成小组进行互动讨论。我认为是个很好的形式。可以在今后继续深化完善。 
　　还有一点建议。培训中，每堂课的时间都比较长，学员容易分散精力，错过精辟的讲解。老师也显得辛苦。可考虑增加并固定课堂中间休息的时间，提高培训的效率。 
　　感谢我们小组的成员和组长邓叶峰，他们在讨论中都提供了很好的帮助。

function SolarTerm(DateGL){
  var SolarTermStr=new Array(
        "小寒","大寒","立春","雨水","惊蛰","春分",
        "清明","谷雨","立夏","小满","芒种","夏至",
        "小暑","大暑","立秋","处暑","白露","秋分",
        "寒露","霜降","立冬","小雪","大雪","冬至");
  var DifferenceInMonth=new Array(
        1272060,1275495,1281180,1289445,1299225,1310355,
        1321560,1333035,1342770,1350855,1356420,1359045,
        1358580,1355055,1348695,1340040,1329630,1318455,
        1306935,1297380,1286865,1277730,1274550,1271556);
  var DifferenceInYear=31556926;
  var BeginTime=new Date(1901/1/1);
  BeginTime.setTime(947120460000);
     for(;DateGL.getFullYear()BeginTime.getFullYear();){
        BeginTime.setTime(BeginTime.getTime()+DifferenceInYear*1000);
     }
     for(var M=0;DateGL.getMonth()>BeginTime.getMonth();M++){
        BeginTime.setTime(BeginTime.getTime()+DifferenceInMonth[M]*1000);
     }
     if(DateGL.getDate()>BeginTime.getDate()){
        BeginTime.setTime(BeginTime.getTime()+DifferenceInMonth[M]*1000);
        M++;
     }
     if(DateGL.getDate()>BeginTime.getDate()){
        BeginTime.setTime(BeginTime.getTime()+DifferenceInMonth[M]*1000);
        M==23?M=0:M++;
     }
  var JQ="";
  if(DateGL.getDate()==BeginTime.getDate()){
    JQ+="今日<"+SolarTermStr[M]+">";
  }
  else if(DateGL.getDate()==BeginTime.getDate()-1){
    JQ+="明日<"+SolarTermStr[M]+">";
  }
  else if(DateGL.getDate()==BeginTime.getDate()-2){
    JQ+="后日<;"+SolarTermStr[M]+">";
  }
  else{
   JQ="";
   if(DateGL.getMonth()==BeginTime.getMonth()){
      JQ+="本月";
   }
   else{
     JQ+="下月";
   }
   JQ+=BeginTime.getDate()+"日<"+SolarTermStr[M]+">";
  }
return JQ;
}

    关于2012年校内文科科研创新计划项目申报工作的通知
    

    [责任编辑]李敏妍　[信息提供]文科处 刘津芳
　　文科各院系（所）、医学院及有关单位： 
　　2012年度上海交通大学文科科研创新计划项目工作今日正式启动，现将有关事宜通知如下：
　　一、指导思想与基本要求。主要资助具有全局性、前瞻性、战略性重大理论价值和实践意义的课题的前期研究；重点资助新兴、边缘、交叉学科课题的前期研究；重点资助有学术积累价值的基础研究课题的前期研究。坚持“看花浇水、择优立项”的原则，把培育大项目和青年人才培养相结合，着力提高我校教师申报高层次科研项目和科研成果的竞争力。本项目不设课题指南，但选题要符合立项的指导思想，作为申请国家社科基金等大项目的预研究，要与重点学科建设结合起来。
　　二、项目类别及申报条件。文科科研创新计划项目共设两个类别：一是特色研究项目，面向全校文科具有高级专业技术职称的在岗教师和科研人员；二是青年培育项目，重点资助39周岁以下具有博士学位的文科青年教师。不具有博士学位的教师申报青年课题需由两名正高职称的同行专家推荐。有在研国家社科基金项目或国家自然科学基金者不得申报今年的“特色研究项目”，承担在研教育部、上海市等省部级以上科研项目未完成者，不得申报今年的“青年培育项目”。已经承担过“特色研究项目”的教师不能再申报该项目，已承担过“青年培育项目”，顺利结项的，符合条件的可以申报“特色研究项目”。
　　三、立项规模与资助额度。本年度计划资助约20项。其中，特色研究项目约10项，每项资助6万元。青年培育项目约10项，每项资助3万元。
　　四、组织申报与材料要求。申报没有名额限制，但是要以院系统一组织申报。院系要对申报者的条件严格审核，认真组织推荐具备条件的教师积极申报。每项需报送院系盖章审核过的《申请书》一式5份，匿名评审活页一式5份；《申请书》及活页用A4双面打印。《申请书》电子版及《申报清单》电子版发送至wkjs@sjtu.edu.cn。
　　五、时间安排与其他事项。本次申报截止日期为2012年10月29日，逾期不予受理。各申报院系请在此之前将审核合格的申报材料盖章后报送文科建设处。
